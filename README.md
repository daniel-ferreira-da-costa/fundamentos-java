# Exercícios de Lógica de Programação

## Exercícios Básicos

1. **Escreva um código que receba o nome e o ano de nascimento de alguém e imprima na tela a seguinte mensagem:**  
   `"Olá 'Fulano' você tem 'X' anos"`

2. **Escreva um código que receba o tamanho do lado de um quadrado, calcule sua área e exiba na tela**  
   Fórmula: `área = lado x lado`

3. **Escreva um código que receba a base e a altura de um retângulo, calcule sua área e exiba na tela**  
   Fórmula: `área = base x altura`

4. **Escreva um código que receba o nome e a idade de 2 pessoas e imprima a diferença de idade entre elas**

## Exercícios Avançados

5. **Escreva um código que receba o nome, ano de nascimento e cidade de três pessoas. O programa deve calcular a idade de cada uma, exibir uma mensagem personalizada para cada pessoa e, ao final, mostrar qual delas é a mais velha e qual é a mais nova.**

6. **Escreva um código que receba o comprimento dos três lados de um triângulo, verifique se eles podem formar um triângulo (usando a regra: cada lado deve ser menor que a soma dos outros dois) e, se sim, calcule e exiba a área usando a fórmula de Heron:**  
   - s = (a + b + c) / 2  
   - área = sqrt(s * (s - a) * (s - b) * (s - c))

7. **Escreva um código que receba o nome e as notas de três provas de dois alunos. O programa deve calcular a média de cada aluno, exibir uma mensagem com o nome e a média, e ao final informar qual aluno teve a maior média (ou se houve empate).**
